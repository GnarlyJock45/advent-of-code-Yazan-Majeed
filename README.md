# advent-of-code 2025 by Yazan and Majeed
This repository includes the combined work of both Yazan Ghassan and Abdulmajeed Alkhalaf towards solving the 2025 advent of code event.

## Motivation
Yazan and Majeed are humble students with humble ambitions who are unlikely to partake in this sort of task, but on one ordinary evening, Yazan was chatting with Dr.Prashant Aparajeya (their previous instructor and now friend). Yazan felt bold enough to ask Dr.Prashant for a reccomendation letter given that both him and Majeed were outstanding students during their course. Unfortunately for both of them, Dr.Prashant made it clear that they had to earn it and hence he tasked them with completing the advent of code 2025 challenge during the span of a week. Sparked with this motivation, both of them set out on tackling this intimidating yet exciting task.

## About the Advent of Code
The advent of code is a project created by Eric Wastl which involves a set of fun, yet challenging programming puzzles. This event is often used as interview prep, university coursework and company training. The set of puzzles is updated every christmas. This repository includes the 2025 set of puzzles.

Link to the event:https://adventofcode.com/
